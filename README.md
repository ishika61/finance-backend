## Live API
https://finance-backend-of5n.onrender.com

## GitHub Repo
https://github.com/ishika61/finance-backend

## Features
- User Role Management (Admin, Analyst, Viewer)
- Financial Records CRUD
- Dashboard Analytics (income, expense, trends)
- Role-based Access Control
- JWT Authentication
- Pagination & Search

## Tech Stack
- Node.js
- Express.js
- MongoDB

## Setup
npm install
npm run dev
